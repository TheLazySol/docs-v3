# API Documentation

Epicentral Labs provides a comprehensive API for developers to integrate with our platform.

## Authentication

All API requests require authentication using your API key:

```typescript
const headers = {
  'Authorization': `Bearer ${API_KEY}`,
  'Content-Type': 'application/json'
};
```

## Base URL

```
https://api.epicentral-labs.com/v1
```

## Endpoints

### Get Markets

Retrieve available trading markets.

```typescript
GET /markets

// Response
{
  "markets": [
    {
      "id": "SOL-USDC",
      "baseAsset": "SOL",
      "quoteAsset": "USDC",
      "active": true,
      "minOrderSize": 0.1,
      "maxOrderSize": 1000
    }
  ]
}
```

### Get Options

Retrieve available options for a specific market.

```typescript
GET /options?market=SOL-USDC

// Response
{
  "options": [
    {
      "id": "SOL-USDC-CALL-100-20241231",
      "type": "call",
      "strikePrice": 100,
      "expiration": "2024-12-31",
      "premium": 5.25,
      "volume24h": 1250.50
    }
  ]
}
```

### Place Order

Place a new options order.

```typescript
POST /orders

// Request Body
{
  "optionId": "SOL-USDC-CALL-100-20241231",
  "side": "buy",
  "quantity": 1,
  "price": 5.25,
  "orderType": "limit"
}

// Response
{
  "orderId": "order_123456",
  "status": "pending",
  "filledQuantity": 0,
  "remainingQuantity": 1
}
```

### Get Order Status

Check the status of an order.

```typescript
GET /orders/{orderId}

// Response
{
  "orderId": "order_123456",
  "status": "filled",
  "filledQuantity": 1,
  "averagePrice": 5.20
}
```

### Get Positions

Retrieve your current positions.

```typescript
GET /positions

// Response
{
  "positions": [
    {
      "optionId": "SOL-USDC-CALL-100-20241231",
      "quantity": 1,
      "averagePrice": 5.20,
      "unrealizedPnl": 0.50,
      "realizedPnl": 0
    }
  ]
}
```

## WebSocket API

For real-time data, use our WebSocket API:

```typescript
const ws = new WebSocket('wss://api.epicentral-labs.com/ws');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Real-time update:', data);
};
```

### Subscribe to Market Data

```typescript
// Subscribe to market updates
ws.send(JSON.stringify({
  type: 'subscribe',
  channel: 'market',
  market: 'SOL-USDC'
}));
```

### Subscribe to Order Updates

```typescript
// Subscribe to your order updates
ws.send(JSON.stringify({
  type: 'subscribe',
  channel: 'orders',
  apiKey: 'your-api-key'
}));
```

## Error Handling

All API responses follow a consistent error format:

```typescript
{
  "error": {
    "code": "INVALID_ORDER",
    "message": "Order quantity must be greater than 0",
    "details": {
      "field": "quantity",
      "value": 0
    }
  }
}
```

### Common Error Codes

- `INVALID_API_KEY`: Invalid or missing API key
- `INSUFFICIENT_BALANCE`: Not enough balance for the order
- `INVALID_ORDER`: Order parameters are invalid
- `MARKET_CLOSED`: Market is currently closed
- `RATE_LIMIT_EXCEEDED`: Too many requests

## Rate Limits

- **REST API**: 1000 requests per minute
- **WebSocket**: 10 connections per API key
- **Order Placement**: 100 orders per minute

## SDKs
